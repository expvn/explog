Các bạn lưu ý, series này là kiến thức cơ bản và rút gọn để học cấp tốc vì vậy sẽ không có những giải thích chuyên sâu, thay vào đó sẽ là giải thích cách sử dụng và ví dụ minh họa ngắn gọn và dễ hiểu. Nhằm mục đích cho các bạn tìm hiểu Python cấp tốc.
**1\. CÀI PYTHON**
**Download:** [Download Python | Python.org](https://www.python.org/downloads/)
![](image-1024x789.png)
Như hiện tại, phiên bản Python là 3.11.4. Bạn bấm vào nút Download Python 3.11.4 để tải về fiel cài đặt python.
Tiếp theo, bạn mở file cài đặt mới tải về
![](image-1.png)
Thực hiện các bước như trong hình: tick chọn 2 dấu tích ở Use admin... và Add python.exe to PATH. Sau đó bấm Install Now. Bạn chờ cài xong thì bấm tắt cửa sổ cài đặt.
**2\. CÀI VISUAL STUDIO CODE**
**Download:** [Download Visual Studio Code - Mac, Linux, Windows](https://code.visualstudio.com/download)
Tùy vào hệ điều hành bạn đang sử dụng mà chọn cho đúng bản. Ở đây mình dùng Windows nên sẽ chọn Windows
![](image-2-1024x761.png)
Sau khi tải xong, bạn bấm vào file vừa tải và làm theo hướng dẫn sau:
![](image-3.png)
Chọn I accept sau đó bấm Next>
![](image-4.png)
Chọn Next>
![](image-5.png)
Chọn Next>
![](image-6.png)
Chọn như hình sau đó bấm Next>
![](image-7.png)
Chọn Install
![](image-8.png)
Chọn Finish và mở Visual Studio Code lên
**3\. Cài Extension cho Visual Studio Code**
Mỏ Visual Studio Code, bạn chọn lần lượt như hình:
![](image-9-1024x683.png)
1 - Chọn biểu tượng Extension ở menu bên trái (Ctrl + Shift + X)  
2 - Gõ khung tìm kiếm: "python"  
3 - Chọn Python (Install)
Sau khi cài đặt xong, bạn bấm vào Menu File> chọn New File
![](image-10.png)
Menu mở xuống, bạn chọn Python
![](image-11-1024x506.png)
Sau đó, bạn save file bằng cách chọn menu File> Save as... (Ctrl + Shift + S)
![](image-12-1024x681.png)
Cửa sổ hiện ra bạn chọn nơi lưu, đặt tên file và bấm save
![](image-13-1024x637.png)
**4\. Hello World**
![](image-14-1024x635.png)
bạn gõ thử như hình
```
print('Hello World')
```
